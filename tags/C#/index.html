委托和事件（一）

委托、事件、Lambda表达式，下一个难关，准备开闯。

什么是委托？

委托是持有一个或多个方法的对象。所以，委托是可以执行的对象，当执行委托的时候，就是执行委托所持有的方法。

举个栗子

有主管和员工两个类，主管类里有个方法是扣钱，员工类里有个方法是偷懒，那么这个时候就可以创建一个委托，即员工一但偷懒，主管就执行扣钱。
